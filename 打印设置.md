# 布丁喵 POS — 收银机静默打印设置（Windows）

让 POS 页面结账时**自动直接出小票、不弹打印框、界面像 App**。这套在店里那台 HP 收银电脑上验证通过。

换电脑、重装、或教店员时，照这份从头做一遍即可。

---

## 一、一次性设置

### 1. 打印机（最关键）
`--kiosk-printing` 是「闷头送到**默认打印机 + 驱动默认纸张**」，所以这两项必须对：

1. Windows 设置 → **蓝牙和设备 → 打印机和扫描仪**。
2. 关掉「**让 Windows 管理默认打印机**」（不关它会自动把默认换走，导致时好时坏）。
3. 点热敏打印机（本店叫 **cashier**）→ **设为默认打印机**。
4. 点它 → **打印首选项 (Printing preferences)** → **纸张大小 (Paper Size)** 设成热敏纸尺寸（**80mm / 72mm**，58 机就选 58mm）→ 确定。

> 验证：设置里 cashier 显示「默认」；直接「打印测试页」能出纸。

### 2. 建 POS.bat（用 .bat，不要用普通快捷方式）
Windows 快捷方式对这种「多参数 + 网址」的长命令经常解析不全（会丢掉 `--app`，开成普通 Chrome 停在 Google）。用 **.bat 脚本**最稳。

1. 打开**记事本**，原样粘贴（Chrome 路径按实际安装位置，本店是 x86）：

```bat
@echo off
start "" "C:\Program Files (x86)\Google\Chrome\Application\chrome.exe" --kiosk-printing --user-data-dir="C:\Users\hp\Documents\poschrome" --app="https://pudding-meow.vercel.app/pos.html?v=4"
```

2. **文件 → 另存为**：保存类型选「**所有文件**」，文件名 **POS.bat**，存到**桌面**。

命令三段的作用：
- `--kiosk-printing` → 打印不弹框、直接出。
- `--user-data-dir="…"` → 开一个**专属独立 Chrome**，保证开关一定生效（不受平时那个 Chrome 开没开影响），也和日常上网互不干扰。
- `--app="…"` → 像 App：没有地址栏、没有标签页。

> Chrome 装在别处时换路径：
> - 按用户安装：`C:\Users\hp\AppData\Local\Google\Chrome\Application\chrome.exe`
> - 找不到就右键任务栏 Chrome 图标 → 右键「Google Chrome」→ 属性 → 看「目标」里的真实路径。

---

## 二、每天怎么用
- 开店：双击桌面 **POS.bat** → 进 POS（像 App）。第一次进要输 PIN（默认 `0000`）。
- 收银、线上 TNG 单进来 → **自动直接出单，不弹框**。
- 想连任务栏也藏掉、全屏：打开后按一下 **F11**（打印照常）。退出全屏再按 F11，关闭窗口 `Alt+F4`。
- 可把 POS.bat「固定到任务栏」，点一下就开。

---

## 三、别乱动（保住成果）
- cashier 保持**默认打印机**、纸张 **80mm**。
- 「让 Windows 管理默认打印机」保持**关**。
- POS.bat 里的命令别改。

---

## 四、排查（万一又不行）
按顺序对：

| 现象 | 多半原因 | 处理 |
|------|----------|------|
| 直接「打印测试页」都不出纸 | 驱动 / USB / 纸张问题 | 先把打印机本身弄通，与浏览器无关 |
| 静默没反应、什么都不出 | 默认打印机被换走，或驱动默认纸张不对 | 重新把 cashier 设默认 + 纸张 80mm |
| 会弹打印框（要手动点） | 开关没生效——多半没用独立实例，或 Chrome 已在跑 | 用 POS.bat（带 `--user-data-dir` 的那条） |
| 开成普通浏览器停在 Google | 快捷方式把 `--app` 丢了 | 改用 POS.bat（.bat 不会丢参数） |
| 印出来是空白 / 变成 PDF | 送到了「Microsoft Print to PDF」 | 把 cashier 设回默认打印机 |

> `--kiosk-printing` 只送到**默认打印机**、用**驱动默认设置**。记住这句，八成问题都能自己定位。
